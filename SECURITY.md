# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

**Please do NOT open a public issue for security vulnerabilities.**

Instead, email [security@privasys.org](mailto:security@privasys.org) with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact assessment

We will acknowledge receipt within 48 hours and provide an initial assessment within 7 business days.

## Scope

This policy covers the Enclave OS Virtual workload launcher, management agent, Merkle tree computation, OID extension handling, and containerd integration. Vulnerabilities in upstream dependencies (containerd, Go stdlib, etc.) should be reported to their respective maintainers.
